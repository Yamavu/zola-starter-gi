This is a Search Page that loads data from an index and searches it using [elasticlunr.js](https://github.com/weixsong/elasticlunr.js)

English ```en``` is supported out of the box. The following other languages are supported via [Lunr languages](https://github.com/weixsong/lunr-languages#lunr-languages--): German, French, Spanish, Italian, Japanese, Dutch, Danish, Portuguese, Finnish, Romanian, Hungarian, Russian, Norwegian. For other languages a custom stemmer would need to be made, not rocket-science but not trivial either. 

Type your query below to search the site.
